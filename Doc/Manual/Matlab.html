XXX.1 Overview

The SWIG implemention was first based on Matlab R2011b.

XXX.2 Command-line options

The swig command line has a number of options you can use, like to redirect its output. Use swig -help to learn about these. Options specific to the Matlab module are: 

$ swig -Matlab -help
...
